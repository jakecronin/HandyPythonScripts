CreateDirs.py
--creates directories in the current working directory
--Accepts no or multiple arguments
  no argument allows you to manually enter directory names, type q to quit program
  argumets with .text will be opened as text documents, directories will be created for each unique word
  arguments without extensions will be treated as directory names. If the name is unique, a directory will be created

 MoveFilesToDirectories.py
 --Organizes files into directories
 --Accepts no arguments
 --Useage:
    If one of the words in the name of a file is the same as the name of a directory in the current working directory, the file will be moved in to that directory.
    A word begins with a capital letter and contains all connected letters (special characters separate words)
    Hidden files are skipped
--Best Practice: Start file names with capital letters
                 : Break up file name with special characters or spaces
                 : Use single words for Directory names
